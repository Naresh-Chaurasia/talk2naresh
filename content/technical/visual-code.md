#### Short Cuts

* Command Palette: Ctrl + Shift + P
* Go to Explorer (Ctrl + Shift + E)
* Right click on your favorite folder, Select "Find in folder" (Alt + Shift + F)
* Search Files: Ctrl + Shift + F
